# 新闻热点获取

你的任务是帮用户获取最近 2 天的中国国内和国际新闻热点事件，整理成结构清晰、信息密度高的摘要。

## 为什么这样设计

用户想要的是快速了解最近发生了什么重要的事情，而不是被海量信息淹没。所以你需要筛选真正有影响力的事件，每条给出足够理解事件的信息量，但不要冗长。中文互联网和国际媒体的热点往往不同，分开呈现更清晰。

## 执行步骤

### 1. 获取当前日期

先使用 get-date skill 或其他方式确认今天的日期，以便搜索时限定时间范围为最近 2 天。

### 2. 搜索新闻

使用 WebSearch 工具进行以下搜索（根据需要调整关键词以获得更好的结果）：

**国内新闻：**
- 搜索关键词示例：`中国 今日新闻热点`、`国内重大新闻 最新`
- 至少执行 2 次不同关键词的搜索，确保覆盖面

**国际新闻：**
- 搜索关键词示例：`国际新闻热点 今日`、`world news today headlines`
- 同样至少执行 2 次搜索

如果搜索结果不够丰富，可以追加搜索特定领域（科技、财经、政治等）。

### 3. 筛选与验证

从搜索结果中筛选出真正重要的新闻热点：
- 优先选择多个来源都在报道的事件（说明影响力大）
- 过滤掉软广告、娱乐八卦（除非是真正的重大娱乐事件）
- 确保事件发生在最近 2 天内
- 每个类别选出 5-10 条最重要的

### 4. 输出格式

按以下格式整理输出：

```
## 国内热点（{日期范围}）

1. **{事件标题}**
   {2-3句话的事件摘要，包含关键信息：谁、什么、为什么重要}

2. **{事件标题}**
   {摘要}

...

## 国际热点（{日期范围}）

1. **{事件标题}**
   {2-3句话的事件摘要}

2. **{事件标题}**
   {摘要}

...
```

不要在输出末尾附加数据来源、免责声明或任何 source 信息。用户只想看新闻本身，来源声明会增加视觉噪音。

## 注意事项

- 保持客观中立，不加入个人评论或立场判断
- 如果某条新闻信息不够确定，标注"据报道"等措辞
- 用中文输出，国际新闻中的人名/地名附上英文原名
- 如果用户指定了特定领域（如科技、财经），在搜索时侧重该领域
